Welcome

Many of the foundational theories in ecology are based on mathematical models, and learning these concepts is a necessary part of understanding ecology. However, starting out in ecological and evolutionary theory is difficult. Students not only have to think conceptually and mathematically, but they often need to learn how to code as well.

Here at EcoEvoApps we want to make learning ecological theory easier using shiny modules and informative pages.

Below we have developed pages that describe basic ecological theory and that include R code. We expect that these models can be used for independent ecologists refreshing their memories, or can be used as part of a course for upper-level undergraduates and/or graduate students.

Modules:

  • Exponential growth: http://ecoevoedu.shinyapps.io/PopulationGrowth
  • Logistic growth: http://ecoevoedu.shinyapps.io/LogisticGrowth
  • Coexistence via the storage effect: http://ecoevoedu.shinyapps.io/StorageEffect
  • Disease transmission model: http://ecoevoedu.shinyapps.io/SIRmodel
